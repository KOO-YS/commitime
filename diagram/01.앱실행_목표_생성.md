
### 1. 앱 최초 실행 및 목표 생성

```mermaidsequenceDiagram
    actor User as 사용자
    participant App as Flutter App
    participant Provider as GoalProvider
    participant Storage as LocalStorage
    participant Notification as NotificationService

    User->>App: 앱 실행
    App->>Provider: loadGoals()
    Provider->>Storage: getGoals()
    Storage-->>Provider: [] (빈 배열)
    Provider-->>App: 목표 없음
    App->>User: 홈 화면 표시 (빈 상태)
    
    User->>App: "+ 목표 추가" 버튼 클릭
    App->>User: 목표 생성 화면 표시
    
    User->>App: 목표 정보 입력
    Note over User,App: - 제목: "운동 30분"<br/>- 타입: 일반<br/>- 요일: 월~금<br/>- 시간: 18:00<br/>- 캐릭터: 교수님
    
    User->>App: "저장하기" 버튼 클릭
    App->>Provider: addGoal(goal)
    Provider->>Storage: saveGoal(goal)
    Storage-->>Provider: 저장 완료
    Provider->>Notification: scheduleGoalNotification(goal)
    Notification-->>Provider: 알람 예약 완료
    Provider->>Provider: notifyListeners()
    Provider-->>App: 목표 추가 완료
    App->>User: 홈 화면으로 이동
    App->>User: "운동 30분" 목표 표시
```
